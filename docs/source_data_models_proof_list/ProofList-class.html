ProofList

ProofList class Null safety Client-Server API Data Models

Store and manage list of ProofEntrys for ClientService.

Most important functions:

  • load proofs from filesystem with crawler function,
  • notify one ClientService,
  • save new proofs.

If serviceId == null, it create/collect proof for whole day.
