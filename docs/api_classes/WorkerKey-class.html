WorkerKey

WorkerKey class Client-Server API

Immutable class to create WorkerKey from/to JSON strings.

api_key - should be unique for each WorkerProfile.

And few parsing methods[]. TODO: implement active server switch.

Example key look like this:

app : "AIS3USON web"
api_key : "3.01567984187"
name : "Test"
worker_dep_id: 1
dep_id : 1
dep : "Test Department"
servers: "https://alexqwesa.fvds.ru:48080|http://alexqwesa.fvds.ru:48081/|https://192.168.0.102:48082"
db : "kcson"
comment : "any text"
certBase64 : ""

Note: https:// or http:// in servers strings is necessary!
