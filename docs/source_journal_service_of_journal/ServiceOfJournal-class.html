ServiceOfJournal

ServiceOfJournal class Null safety Journal

ServiceOfJournal is a class to store one entry of Journal (one input service).

It is created in state ServiceState.added, with current, date and unique uuid.

Journal send services in states ServiceState.added and ServiceState.stalled to DB, and marked them ServiceState.finished or ServiceState.rejected.

Services in state ServiceState.finished or ServiceState.outDated send to Journal.hiveArchive on next day.
